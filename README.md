Este repositorio contiene únicamente el código utilizado para realizar diferentes consultas en MongoDB a partir de una colección basada en jugadores del Mundial. Las consultas incluyen operaciones básicas (insertar, actualizar y eliminar documentos) y otras un poco más avanzadas, como filtros combinados y agregaciones para obtener estadísticas específicas.

El objetivo es servir como una referencia sencilla y práctica para quienes quieran ver ejemplos reales de cómo consultar datos en MongoDB usando una colección con información variada.
